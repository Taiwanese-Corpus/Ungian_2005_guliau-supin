
 
薰害
L.T.K ek8.
I.薰e5款式。
1.薰e5來歷。西班牙e5 gau5人科倫布，chhe7 著亞美利加州e5 時，才起頭看著hit款e5物。Hit所在e5生番有teh食薰。Koh無偌久，有一個英國e5大臣去hia，續帶兩項e5物倒轉，就是薰kap蕃薯。今這兩項款e5厲害是大各樣。薰是第一陷害人；蕃薯是第一利益人。生番本然扑算食薰是會醫好人e5病，到路尾就知食薰是hou7 人e5病較傷重，可惜文明國e5百姓，去趁野蠻人e5歹風俗，致到萬國e5人大半已經入這個羅網。
2.薰e5食法。( 1 ) 用suh--e5:是平常e5法度。 ( 2 ) 用食--e5:是親像台灣人食檳榔e5款；che是第一粗俗e5法度。( 3 ) 用sngh--e5:是用薰沫來sngh ti7 鼻空；這款現今罕tit用。
3.薰e5賣法。( 1 ) 歸梱賣hou7 會社。( 2 ) 熟芽薰，是將幼幼e5葉來做，算是第一好，第一貴--e5。( 3 ) 薰枝，就是用薰葉屎chham毒藥( Opiate ) 來做。( 4 ) 條絲薰。Ti7這四款中最第一大害e5是薰枝。若食這款較久，到路尾腦筋會麻痺，失精神，致到人boe7盡本份。
4.薰e5毒。醫生試驗了，就知薰是毒物，才出明明e5憑據。人e5胃常常ai3脫離薰e5毒，致到腦髓燒，頭殼眩，人續漸漸憨，面歹色。胃an2-ni濟濟擺試看會脫離這個毒物，到路尾無力續降伏伊做奴才。
5.薰e5質。( 1 ) 尼戈疹：是大害e5毒物。( 2 ) 炭酸：是hou7 人頭殼眩，續愛睏。( 3 ) 炭丹沙：che是hou7 人e5肉筋會chhoah，續害ti7 心臟。( 4 ) 亞mou里仔：是會麻人e5舌，也會hou7 出瀾 e5筋緊礁，致到所食e5物boe7利益身軀。
II.禁薰e5因端。    
1.薰 m7 是米糧，ti7 薰e5質無一項會利益人e5身軀。    
2.薰會連累人，致到做薰e5奴才，真oh 得著thau2放。
3.薰會阻擋五臟六腑kap四肢五官e5功，續hou7身軀快tu著病疼。
4.Ti7 應接中來捷捷食薰，不但有失禮；koh會hou7無食薰e5人鼻hit個味來真艱苦。
5.食薰是做少年人e5歹模樣。
6.薰是極la5-sam5 e5物，會hou7嘴齒烏，hou7 嘴歹味，來hou7 kap醫講話e5人艱苦。
7.扑損錢銀。    
III. 薰e5害
有害ti7 囡仔
1.囡仔年歲漸漸大，百體軟chiaN2，若常常食薰枝會阻擋伊e5長成，boe7 tit thang勇壯，身軀boe7 到自然e5大漢。念頭boe7 gau5 想，逐項快boe7記tit，實在boe7 gau5 讀冊。囡仔食薰，腦筋會漸漸鈍，伊那久會那lun2-neh，伊e5 志氣逐日那軟弱，也tu著甚mih代誌，伊真快受氣。Koh一項，囡仔食薰比無食薰e5囡仔較無禮e5款式，無有孝序大人，也無降服ti7 伊主人。Che是gau5 e5醫生研究了，所出--e5憑據。
2.有害ti7 大人。Che是甚mih因端？( 1 ) 是因為食熟薰e5人teh家己騙，teh扑損熟薰無害。( 2 )熟薰e5害，boe7忽然間顯明；m7-ku2伊e5 毒物漸漸teh入ti7 伊e5 腦筋肉筋，到路尾愛改，志氣e5力無夠額。( 3 ) 因為看見別人hiah濟teh食，續teh做禮數e5路用。
食薰 m7 准百體來做全功，真快tu著破病，boe7 gau5抵擋艱苦事，較快失志；較快降伏ti7別款e5歹癖，親像食酒，食嗎啡hiah-e5 類。食薰e5人伊e5 嘴舌常常礁，伊e5 肺hou7 薰煙害，就較gau5喘；m7-ku2第一大害，就是心臟快著薰害e5心病。
有害ti7 家眷。( 1 ) 了錢，真大扑損，就是了某子所應該食，穿，用e5錢。( 2 ) 家己e5厝內不時有薰e5毒味，連累家內e5人，家己teh食，家內e5人快趁伊e5 模樣；伊無力thang勸勉別人。( 3 ) 有時婦人人身軀軟弱，若是丈夫食薰真濟，伊e5 病真oh創；有時醫生苦勸丈夫著禁食薰，伊e5 某才好。( 4 ) 生子e5關係。食薰家己e5身軀軟弱，會連累子兒也軟弱；che是自然有hit個遺傳性e5關係。An2-ni chiah e5子兒，無力抵擋世間e5引誘迷惑，致到伊koh較快趁逐等號 e5 歹；是伊e5 志氣，伊e5 心神，伊e5 氣力，攏已經lam2。子兒來陷落ti7 這款e5罪過，做食薰e5序大人有大見笑。
列位同僚iau2 teh食薰e5傳教師，lin2讀這篇了，若koh無隨時改，來扑滅你本身食薰e5歹，就你beh怎樣教示教會著改逐項e5歹--ah？
