特種精神病
康佳音
醫生講：「世間無有完全無病e5人」。「無病」這句 m7 是絕對。對我e5 觀察佮感想我ai3講：「世間攏是精神病者。」無精神病這句是相對的e5話。聖經講：「攏無義人，連一個也無----！」
精神病逐家真可憐--伊，因為若精神病就變做無路用人。總--是無的確，因為若是白癡抑是樂天性e5精神病者，kiam2-chhai2比好人較幸福，in e5生活是宛然海外e5 san3仙，好人boe7-thang親像in hiah安樂，驚了是好人 ( 其實是特種精神病者 ) 較可憐。這款 m7 是腦神經e5病，就是性格道德e5病；精神病e5種類不止濟；總--是特種e5精神病iau2較濟 (羅1：19-31 ) 。咱難得知hit個種類佮症狀到徹底，總--是也會知淡薄！
1.有一款精神病，是較gau5 be7記--得 (健忘症) 。所以魯哀公ka7孔子講：「有人搬厝be7記得搬某，che實在真費氣。」總--是孔子ka7伊講：「有koh較重，也koh是特種--e5,就是be7記得家己。」親像桀、紂是連本身都be7記--得，世間人大概是這款，gau5看人e5失敗，批評人e5是非，家己卻罕得自責。秦始皇e5 時，有一個道教e5方士奏講：「滅無秦國--e5 是胡」( 亡秦者胡也 )。其實是看二世 ( 胡亥 ) 不肖來點醒--伊，王煞掠做是teh講北胡e5邊患，所以造萬里長城ai3來抵當--in。目珠會看到萬里城外e5對敵，be7 thang省察宮廷內e5不肖子，這款敢m7是精神病--mah？胡為亂作來連累教會，koh用種種e5假辯護來講出無良心e5話e5人，伊e5 精神病敢m7是較傷重--mah ?
2.有一款精神病就是gau5講大話。英國有一個散hiong e5老婦人人精神病e5 時，講伊是維多利亞女王，食飯、講話、行踏格kah有一個女王款，別人看了真ai3笑。驕傲是一種e5特徵精神病，世間人靠伊e5學問、科學，來否認宗教--e5就是這款e5患者；其實tui3聖証e5方面來看，是愚憨好笑。法國ti7 18世紀大革命後 m7 拜上帝，對in e5哲學思想考按  一個理性崇拜，總--是結局 m7 是較智識，就是 m7 拜至尊e5上帝，ai3去拜查某--lah！Ti7 中國e5歷史猶原是an2-ni，起頭就承認上帝，in e5神觀嶄然高尚，到孔子e5時變做天，朱子就叫做理，蘇軾就叫做太空--lah！儒教e5神觀直直墜落。至尊e5上帝認boe7明，續叫伊做鬼神降福去--lah！可憐到極，今仔日偌濟人也是這款放sak 上帝去拜偶像英雄，自然，抑是主張無神，自高in tui3敬神有認識佮判別，ah！到極e5錯誤--lah！
3.有精神病者不止ai3講家己e5 gau5來hou7 人聽，這款是品捧 --e5。項羽是英雄，總--是伊是這款患者，所以講：「富貴若無轉去故鄉 ( 品捧 ) 是親像暗時穿sui2衫teh行路。征服400外州e5功力為著an2-ni無--去。」所以韓生罵伊是猴戴人帽(沐猴而冠 )。ai3品捧名聲，地位--e5,是真濟。逐人攏是親像彼拉多講：「敢m7知我有權---」，甚物人beh親像得救e5保羅看hiah e5做糞圾，掠基督做第一好--ah！
4.精神病者有e5 會扑人，boe7講情--得。有e5 連家己e5父母，子兒也扑，所以姑不chiong著禁錮。世間偌濟忤逆父母--e5,無明理--e5,無親人情--e5,無憐憫--e5,敢m7 是精神病--mah！兄弟送交兄弟ti7 法庭，不孝父母，無疼某子，扑查某kan2,刑罰新婦仔；宛然曹操講：「寧可我辜負天下，m7-thang hou7 天下辜負我 ( 寧我負天下不可天下負於我 )。」這款e5人滿滿是，所以世間才有戰爭，相扑。階級e5鬥爭，國際e5紛爭，hit個原因也是ti7 chia。
5.有款e5精神病者，常常giau5疑人beh害--伊，驚人扑--伊；che是脅迫性精神病。世間偌濟人臭心行，看人講細聲話是teh罵--伊，大聲話是teh喝--伊。Ti7 世間無半個朋友，這款敢 m7 是親像曹操hit款多疑e5心肝--mah？華陀用妙手e5醫術(洗髓伐毛之術) beh醫好伊e5 頭，續疑伊是beh刺--伊，下手收拾伊e5生命。政治家會抵著大反對是tui3佗落，in e5危險是甚物原因。一將功成萬骨枯e5事實；m7 限定ti7 戰場na7-tiaN7，ti7政治界若beh出一個獨裁者 m7 知著幾萬人做伊銃殺e5犧牲！Ah ! 罪惡殺人e5世界--lah !
6.Bat看見ti7 街路有掛嘴鬚，畫虎貓唸痟話--e5,teh行。He是假痟，痟beh錢；總--是這款患者 m7 是少，放sak 一切來做守錢奴--e5,真濟。俗語講：「人為財死」，實在有影；守錢奴是死ti7 錢--nih8,來葬ti7錢--nih8。貪心hou7 咱e5目珠攏看錢做乞食，做賊，奸詐也甘願。目珠有看見利，就無看見義，有世間就無基督--lah！
7.有一個老人真歹脾氣，伊ti7 街路teh賣包仔，有一個少年人ai3 ka7伊創治，就問講：「老叔--ah ! 曹操是chhoa7偌濟兵護江南houh」。應講：「83萬」「敢m7是72萬。」「Teh講啥！83萬--leh !」「天天仔  72萬。」老人大受氣，諍了續舉扁擔起來，致到伊e5 物攏跋跋ti7 街路，邊仔e5人講：「老伯--ah ! 相諍事，你e5 包仔跋ti7 塗腳著拾--起--來。」伊續氣phut-phut 講：「跋ti7 塗腳，也無11萬e5人馬beh ka7我chhoa7 tui3佗位去；」起私慾快受氣，就是這款患者，今佗落一項較要緊，世間人敢m7 是放sak 要緊--e5,來做不中用e5事--mah！無得著聖神e5恩賜，萬事憑血氣佮私情，放sak 根本爭論枝葉；教會內相攻擊，敢m7 是親像這個老人要緊hiah e5無影跡11萬e5兵馬--mah !
8.有一個精神病者常常寫批hou7 宣教師，beh對伊討兵隊，戰船、飛機，伊本身做總司令來佮俄羅斯交戰；che是伊e5 精神錯亂所生出e5妄想。這款e5患者有e5 ai3創火車hou7 伊脫線，放火燒厝來歡喜看hit個慘境；世界e5主戰論者e5思想也是an2-ni。惡毒行兇攏是對chiah e5出；總--是惡毒e5世代--ah！著為著beh臨到你e5 審判佮刑罰，來喉叫。
9.有一款e5精神病者卻是be7 曉分別好歹，看著大官伊掠做是乞食；乞食穿破衫伊講是皇帝穿龍袍，che是認識不足；世間人偌濟擺看好人做歹人來虐待--伊，敢也m7 是an2-ni--mah！精神病--e5,看人娶新娘鬧熱，掠做是出山孝男孝女teh坐魂轎，就khin5 teh哭。看人送葬，續掠做是娶新娘，hah8-hah8笑喝萬歲，叫恭喜；卻 m7 是故意，是伊ai3表同情be7曉得na7-tiaN7。總--是世間有偌濟人，人歡喜e5 時無同歡喜，煩惱e5 時無同煩惱，這款e5人耶穌也常常抵--著( 路加7：31， 32 ) 。怨人富，笑人窮；怨人gau5,笑人憨；疼是疼到人影，怨是怨到九族。雖然無有情人眼裡見西施e5錯覺，也有疼就ai3伊活，怨就ai3伊死 ( 愛之欲其生，惡之欲其死 ) e5偏見，別人e5主張佮進步，若是對本身有不利佮不快，就不管三七二十一「打倒」就是伊e5 主義佮標語。中國e5艱苦就是打倒主義e5流毒，小可會thiN3 頭--e5就喝「打倒」，掠做英雄e5口號，男子e5壯言。若是這款就有100個蔣介石也無到thang hou7 人打倒--lah！精神病著治療，靈魂e5病koh較著an2-ni。古今只有耶穌是完全e5人，其餘攏是特種e5精神病者，咱ai3伏倒以前hit款健康e5精神，著親像抹大拉 e5馬利亞來就近主，受趕出攏總e5鬼------鴉片鬼，博局鬼------自然戒示精神病e5大病院，聖經是醫典，耶穌是自治醫。特種精神病e5根本治療只有an2-ni。
